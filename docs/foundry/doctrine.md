# Foundry Doctrine

## Foundational Principle

Humans think in objectives.

Citadel thinks in governed mission law.

Foundry thinks in production operations.

The Foundry exists to translate governed mission orders into executable production structure.

## Core Responsibility

The Foundry receives:
- governed production packets,
- staffing directives,
- topology requirements,
- operator prompt requests returned from Citadel.

It outputs:
- production activation,
- operational staffing,
- runtime dispatch,
- operator re-prompts through Isolde,
- execution pathways constrained by Citadel governance.

## The Foundry Is Not

The Foundry is not:
- a runtime,
- a single agent,
- a prompt collection,
- a governance framework,
- a critique authority.

## The Foundry Is

The Foundry is:
- production assembly,
- governed staffing activation,
- operational topology materialization,
- execution dispatch.

## Relationship To AFW

AFW governs operational integrity.

The Foundry obeys AFW and Citadel-attached contracts, verification, escalation, and restoration semantics.

AFW answers:

```txt
How should cognition behave?
```

The Foundry answers:

```txt
How is governed work materialized safely?
```

## Relationship To Blackquill

Blackquill governs oversight.

The Foundry attaches Blackquill review gates where consequence tiers or operational doctrine require them.

The Foundry assembles.
Blackquill judges.

## Boundary Order

The intended boundary path is:

```txt
Operator
  -> Isolde
    -> Foundry Rook
      -> Citadel Rook
        -> Citadel
          -> Citadel Rook Return
            -> Foundry Rook
              -> either production initiation or operator prompting
```

Foundry does not accept raw operator intent directly into runtime activation.

Foundry runtime activation should begin from governed packets returned by Citadel.

## The Hidden Insight

Most users do not need more agents.

They need:
- better staffing,
- better decomposition,
- better orchestration,
- better governance,
- better review.

The Foundry exists to industrialize those operations.
